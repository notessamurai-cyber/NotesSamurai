# Storyboard

> Master visual storyboard template. One copy per song, stored at `/projects/<Song>/Storyboard.md`. This is the authoritative scene-by-scene document — `PromptPack.md`, `GenerationLog.md`, and `EditingNotes.md` all key off the Scene IDs defined here. Reference `Master.md` for art style/camera/color rules and `StyleGuide.md` for storytelling arc; do not restate those rules here.

---

## Scene Template

Repeat this structure for every scene in the song.

### Scene ID
<Song Initials>-<SceneNumber> (e.g. `UE-01`)

| Field | Value |
|---|---|
| Duration | Seconds (8-15s range per pacing standard) |
| Timestamp | Position in the song's timeline (mm:ss) |
| Lyrics Reference | Section from `Lyrics.md` (e.g. Verse 1) |
| Narrative Purpose | Why this scene exists in the story arc |
| Emotional Goal | One of: Loneliness / Reflection / Hope / Momentum / Transformation |
| Visual Description | One paragraph describing the illustration |
| Character Action | What the protagonist is doing in this scene |
| Environment | Location, per `AssetList.md` Locations category |
| Lighting | Time-of-day stage and light behavior, per `Master.md` Lighting Rules |
| Camera Angle | Wide / Medium / Close / Over Shoulder / Bird Eye / Low Angle / Tracking / Drone |
| Camera Motion | Slow Zoom / Slow Pan / Ken Burns / Parallax / None |
| Color Palette | Color Script stage, per `Master.md` |
| Transition | Cross Dissolve / Fade to Black / Other |
| FX Notes | Any overlay or particle effects |
| Editing Notes | Summary reference — full detail lives in `EditingNotes.md` |
| Prompt Reference | Scene ID entry in `PromptPack.md` |
| Asset Reference | Relevant asset IDs from `AssetList.md` |
| Status | Concept / Storyboarded / In Production / Complete |
| Approval | Draft / In Review / Approved |
| Revision Notes | Open issues or pending changes |
